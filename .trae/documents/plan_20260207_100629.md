# 修复设置页面问题

## 问题分析

1. **标题修改**：设置页面的标题显示为"个性设置"，需要改为"设置"
2. **退出登录功能**：退出登录按钮点击后没有任何反应，需要添加退出登录功能

## 解决方案

### 1. 修改设置页面标题

- **修改文件**：`pages/Settings.tsx`
- **具体操作**：找到第22行，将"个性设置"改为"设置"

### 2. 添加退出登录功能

1. **修改 Settings 组件接口**：
   - **修改文件**：`pages/Settings.tsx`
   - **具体操作**：在SettingsProps接口中添加onLogout属性

2. **修改退出登录按钮**：
   - **修改文件**：`pages/Settings.tsx`
   - **具体操作**：为退出登录按钮添加onClick事件处理

3. **添加退出登录处理函数**：
   - **修改文件**：`App.tsx`
   - **具体操作**：添加handleLogout函数，实现以下功能：
     - 清除localStorage中的token
     - 设置isAuthenticated为false
     - 导航到登录页面

4. **传递退出登录函数**：
   - **修改文件**：`App.tsx`
   - **具体操作**：在渲染Settings组件时传递handleLogout函数

## 预期结果

- 设置页面的标题将显示为"设置"
- 点击退出登录按钮后，用户将被登出并导航到登录页面
- 再次访问应用时，需要重新登录
