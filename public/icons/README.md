# PWA图标目录

此目录用于存放PWA应用所需的图标文件。

## 图标要求

需要添加以下尺寸的图标文件：

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## 图标生成工具

可以使用以下工具生成不同尺寸的图标：

1. [RealFaviconGenerator](https://realfavicongenerator.net/)
2. [Favicon.io](https://favicon.io/)
3. [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator)

## 图标设计建议

- 使用平台的Logo或品牌标识
- 保持图标简洁明了
- 使用透明背景
- 确保在不同尺寸下都清晰可辨
- 符合平台的视觉风格

## 注意事项

- 所有图标文件必须是PNG格式
- 文件名必须严格按照要求命名
- 图标文件大小应适中，避免过大影响加载速度
- 确保图标在浅色和深色背景下都能清晰显示
